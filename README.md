# Simple HR Candidate Email Response Tool

## Project Overview
This is a simple, beginner-friendly web application designed for HR departments. It allows HR personnel to easily send pre-defined acceptance or rejection emails to candidates. The application uses HTML, CSS, JavaScript, and PHP, along with the PHPMailer library to handle the actual sending of emails securely.

## Folder Structure

HR-Email-Response/
│
├── index.php             # The main HR form to enter candidate details
├── preview.php           # Shows a preview of the populated email before sending
├── send.php              # Backend script that uses PHPMailer to send the email
├── success.php           # Displays success or failure messages after sending
├── style.css             # Styling for the application (clean HR dashboard look)
├── script.js             # Client-side validation and loading states
├── config.php            # Stores your SMTP email credentials securely
│
├── templates/            
│   ├── selected.php      # Email template for selected candidates
│   └── rejected.php      # Email template for rejected candidates
│
├── vendor/               # Folder generated by Composer containing PHPMailer
│
└── README.md             # This documentation file
```

## Installation and Setup

### 1. How to install XAMPP
XAMPP is a free local server environment.
1. Download XAMPP for your operating system from `apachefriends.org`.
2. Run the installer and follow the on-screen instructions (installing Apache and PHP is necessary).
3. Once installed, open the XAMPP Control Panel.

### 2. How to start Apache
1. Open the XAMPP Control Panel.
2. Click the "Start" button next to "Apache".
3. Wait until the background of "Apache" turns green, indicating the local server is running.

### 3. Placing the Project in XAMPP
1. Navigate to your XAMPP installation folder (usually `C:\xampp` on Windows).
2. Open the `htdocs` folder.
3. Place this entire project folder (`HR-Email-Response`) inside the `htdocs` folder.

### 4. How to install Composer
Composer is a dependency manager for PHP, needed to easily install PHPMailer.
1. Download Composer from `getcomposer.org`.
2. Run the installer. It will automatically detect your PHP path from the XAMPP installation.
3. Finish the setup.

### 5. How to install PHPMailer
1. Open your computer's terminal (Command Prompt on Windows, Terminal on Mac).
2. Navigate to your project directory inside htdocs:
   `cd C:\xampp\htdocs\HR-Email-Response`
3. Run the following command to install PHPMailer:
   `composer require phpmailer/phpmailer`
4. This will create the `vendor/` folder automatically.

### 6. How to configure Gmail SMTP
1. Open `config.php` in your text editor.
2. You will need an "App Password" from Google (your normal Gmail password won't work for security reasons).
3. Go to your Google Account > Security.
4. Enable "2-Step Verification" if it's not already on.
5. Search for "App passwords" in your Google Account settings.
6. Generate a new App Password (it will be a 16-character code).
7. Copy this code into the `config.php` file in the `'password'` field.
8. Enter your Gmail address in the `'username'` and `'from_email'` fields.

### 7. How to test locally
1. Open your web browser.
2. Type `http://localhost/HR-Email-Response/index.php` into the address bar and press Enter.
3. Fill out the form, click "Preview", then click "Send Email".
4. Check the success message and verify the email was received by the candidate address you entered.
